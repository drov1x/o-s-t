### 2.1.2 字数据大小
字长 -> 指针数据标称大小
字长影响的最重要的参数就是虚拟内存空间的最大大小
##### 32位程序编译
```shell
gcc -m32 -o test test.c
```
##### 64位程序编译
```shell
gcc -m64 -o test test.c
```
程序运行的具体字长由编译时确定，而非机器确定

##### C语言数据类型大小（字节）
|有符号|无符号|32位|64位|
|--|--|--|--|
|(signed) char|unsigned char|1|1|
|short|unsigned short|2|2|
|int|unsigned int|4|4|
|long|unsigned long|4|8|
|int32_t|uint32_t|4|4|
|int64_t|uint64_t|8|8|
|char *|-|4|8|
|float|-|4|4|
|double|-|8|8|

### 2.1.3 寻址与字节顺序

##### 大端法与小端法
以 `0x01234567`（4字节整数）为例：

|字节顺序|地址0x100|地址0x101|地址0x102|地址0x103|
|--|--|--|--|--|
|大端法|01|23|45|67|
|小端法|67|45|23|01|
